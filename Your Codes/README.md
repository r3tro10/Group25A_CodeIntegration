# Group25A_CodeIntegration
In this formative task you will work with your group to integrate your individual SwiftBot implementations into one cohesive working solution.

As part of Assignment 3 – CS1814 Software Implementation, each group member is required to develop an individual Java program. For the Code Integration Task, you must then work collaboratively as a group to integrate the code developed by all (or a subset) of your group members into a cohesive application that runs as one unified program

For example, if there are 8 active members in the group, the integrated solution should display all 8 tasks on the screen as a menu. A user should then be able to select any one of the tasks from the menu and execute it within the same unified program.

This task provides you with a valuable opportunity to apply key software engineering principles and implement a working solution as a team. You are therefore encouraged to first prepare a basic flowchart to outline the overall program logic and a UI design to show how
users will navigate the integrated program. Once this is agreed within the group, you should proceed to implementation and carry out thorough testing. You may also use GitHub to support systematic development and version control throughout the task.

What is required for this task is to integrate all SwiftBot tasks into one unified program. Your solution may also include additional features beyond the minimum requirement and these will be considered as part of your overall contribution to the task.

In terms of the scope of this formative task, the focus will be on the quality of the integrated program as a whole. The individual tasks will not be re-evaluated, as these are assessed separately as part of Assignment 3. Therefore, if all the required functionalities of a particular
task has not been implemented that would not affect this task. However, each task must be integrated accurately so that it can be selected from the menu and executed correctly within the unified program
